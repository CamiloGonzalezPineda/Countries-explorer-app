#  Countries Explorer App

A modern web application that consumes a public API to display real-time country data, with dynamic search, filtering, and optimized rendering.

---

## Link
https://countries-explorer-app-api.netlify.app


---


---

##  Key Features

*  Real-time search by country name
*  Filter by region
*  Filter by population
*  Optimized input handling with debounce
*  Clean data formatting (population, structure)
*  Dynamic DOM rendering
*  Empty state handling (no results found)
*  Smooth UI animations

---

##  Tech Stack

* HTML5
* CSS3 (Responsive Design)
* JavaScript (Vanilla)
* Fetch API

---

##  API

* REST Countries API
